# Models Directory

## What Goes Here

This folder stores **trained PyTorch model weights** (`.pt` files) generated by the PINN training pipeline (Role 8).

### Expected Files

| Filename | Generated By | Description |
|----------|-------------|-------------|
| `pinn_model.pt` | Role 8 (Training Operator) | Final trained PINN model state dict |

---

## How to Save Model Weights

In your training script, save the model using PyTorch's standard method:

```python
torch.save(model.state_dict(), "outputs/models/pinn_model.pt")
```
