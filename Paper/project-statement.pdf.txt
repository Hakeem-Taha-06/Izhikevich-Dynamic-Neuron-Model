Dear students,  
• Students can work on the project in groups of 8-10 students . Please clearly 
indicate the contribution of each group member . 
• For the course projects, you are required to pick a problem in biomedical 
engineering, show how it can be modeled by ordinary differential equations , 
implement two numerical scheme s for solving these equations, and 
implement an equivalent machine learning (or deep learning) scheme  for 
solving these equations.  
• More specifically, you need to  do the following :
