1. Model Overview
TheIzhikevich Dynamic Neuron Model (DNM) is a two-dimensional reduced neu-
ron model due to Izhikevich [1], as presented in Chapter 4 of Schiesser (2014) [2]. It was
introduced as a computationally efficient alternative to biophysically detailed models such
as Hodgkin–Huxley and FitzHugh–Nagumo, yet retains the ability to reproduce a broad
spectrum of experimentally observed cortical neuron firing patterns, including regular
spiking (RS), fast spiking, intrinsically bursting, chattering, and low-threshold spiking
modes.
The model consists of two coupled first-order autonomous ODEs : the membrane
potential v(t) and the recovery current w(t). A nonlinear voltage equation captures
positive feedback and spike initiation; a linear recovery equation captures slower negative
feedback. An applied input current I(t) acts as a time-varying forcing function.

5. Initial Conditions
Each first-order ODE requires one initial condition (Schiesser 2014, p. 197):
v(0) = v0=−60 mV (resting potential vr) (6)
w(0) = w0= 0 pA (quiescent state) (7)
Time domain: t∈[0,1000] ms with 1001 uniform output points ( h= 1 ms).
6. Nullcline Analysis
Nullclines are curves in the ( v, w) phase plane along which one of the derivatives vanishes.
Their intersections define the fixed points (equilibria) of the system.
6.1v-Nullcline ( dv/dt = 0)
Setting eq. (1) to zero:
w=k(v−vr)(v−vt) +I(t) (8)
This is an upward-opening parabola in the ( v, w) plane. Its vertex lies at v∗=
(vr+vt)/2 =−50 mV. Increasing Ishifts the parabola upward , which can eliminate fixed
points and trigger sustained spiking (bifurcation to limit cycle).

6.2w-Nullcline ( dw/dt = 0)
Setting eq. (2) to zero:
w=b(v−vr) (9)
This is a straight line through ( vr,0) = (−60,0) with slope b=−2. Because b <0, the
line has a negative slope, meaning wdecreases as vincreases above rest — this enables
re-excitation.
6.3 Fixed Points
Fixed points ( v∗, w∗) are found by solving eqs. (8) and (9) simultaneously:
k(v∗−vr)(v∗−vt) +I=b(v∗−vr) (10)
ForI= 0 (sub-threshold), the system has a stable fixed point near ( vr,0) = (−60,0)
— the resting state. For I= 70 pA (supra-threshold), this fixed point may be destroyed
or become unstable, and a stable limit cycle (periodic spiking) emerges.
7. System Dynamics and Coupling
The two ODEs are bidirectionally coupled through v:
Eq. (1):The nonlinear term k(v−vr)(v−vt) drives rapid upswing (spike) when
v > v randv > v t. The recovery term −wdamps excitation. I(t) provides external
forcing.
Eq.(2):The cross-coupling term ab(v−vr) links vtowdynamics. With b=−2,
this term is negative when v > v r, causing wto decrease and thereby reducing
inhibition in eq. (1) — enabling re-excitation.
The net result is a relaxation oscillator : rapid spike in v→reset→slow recovery in
w→re-excitation →repeat. This produces the limit-cycle behavior visible in the ( v, w)
phase plane.
8. Bifurcation Analysis
9. Modeling Assumptions
1.Spatial homogeneity (lumped model): The neuron is treated as a single com-
partment with spatially uniform vandw. No diffusion; only time is the independent
variable (ODEs, not PDEs).
2.Phenomenological dynamics: The model is a two-dimensional reduction de-
signed to reproduce observed spiking behavior, not a biophysically derived conductance-
based model. waggregates multiple ionic currents into one slow variable.
3.Prescribed input current: I(t) is an externally specified step function, indepen-
dent of any state variable — constituting open-loop forcing.
4.Instantaneous spike-reset: Action potentials are not resolved at the integration
timescale. When v≥vpeak, state variables are instantaneously reset via eqs. (3)–(4),
producing essentially discontinuous derivatives.

10. Notes for Numerical Implementation
Step size: h= 1 ms over t∈[0,1000] ms (1001 points). Sharp spikes require a
small step size.
Spike-reset logic: After each step, check v≥vpeak= 35 mV. If true: v[i]←35
(visualization), then v[i+ 1]← −50,w[i+ 1]←w[i+ 1] + 100. This logic is outside
the ODE RHS function.
Non-smooth derivatives: Spike resets produce effectively discontinuous deriva-
tives, limiting achievable convergence order for high-order methods.
Input current array: I(t) is stored as a time-indexed array, requiring the loop to
pass the correct index to the ODE routine.
ODE routine interface: neuron(t, y) takes current time tand state vector
y= [v, w]⊤, returning derivative vector [ dv/dt, dw/dt ]⊤.
